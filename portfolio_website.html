Web Development

As an aspiring game developer, diving into this website project was one of the biggest challenges I set myself. My goal was to create a site that was not only functional, but also had a unique look and feel to showcase my web development skills. To be honest, my knowledge in this area was quite limited to begin with, but the journey has been an incredibly valuable learning experience.

The JavaScript files I've conjured up are perhaps best kept a secret, safely hidden where only the bravest of coders dare to venture. Despite the occasional code-induced headache, the project has turned out quite well and has opened up many opportunities for improvement in both frontend development and graphic design. Overall, this endeavour has significantly improved my skills and given me new insights into these areas.
